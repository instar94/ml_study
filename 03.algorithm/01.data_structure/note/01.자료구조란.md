>  자료구조 : 컨테이너 자료형의 데이터 구조

# 자료형의 구조 


- 리스트
```python
student = ["A","B","C","D"]
```
- 튜플 
	- 한 번 입력되면 변경이 불가 
```python
student = ("A","B","C","D")
```
- 딕셔너리
	- Key and Value
```python
student = {"A":1,"B":2,"C":3,"D":4}
```
- 세트
	- 중복 데이터 허용 X 
```python
student = {100,150,90,110}
```
